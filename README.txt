After you have openned the shopping_cart_version_4.0 directory, do the following:



(1) Open the file named shopping_cart_version_4.0/config.php and fill in your database connection settings

(2) Open the file named shopping_cart_version_4.0/TABLE.sql, copy the query to your server and
    run it to create the required table for this system

(3) Upload the directory named shopping_cart_version_4.0 and its entire content to your server, location this directory through
    your web browser to see the system and you are done.
